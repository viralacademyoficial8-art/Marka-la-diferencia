# Instrucciones para agregar el logo

1. **Descarga la imagen** que compartiste (logo de Makers)
2. **Colócala en la carpeta** `/images/` con el nombre: `logo-makers.png`
3. **Asegúrate que sea PNG o JPG** para máxima compatibilidad
4. **Tamaño recomendado**: La imagen debe ser al menos 300x150px (el navbar la mostrará a 65px de alto)

Si la imagen tiene fondo transparente es aún mejor para lucir profesional.

Una vez que agregues la imagen, commit con:
```bash
git add images/logo-makers.png
git commit -m "🎨 Agregado logo Makers al navbar"
```
